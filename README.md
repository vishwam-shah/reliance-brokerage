# Reliance Brokerage — Next.js 16 Modern App

**A production-grade institutional business brokerage platform built with Next.js 16, TypeScript, Tailwind CSS, and Framer Motion.**

---

## 📋 Project Overview

Modern, accessible, and performant web application for Malaysia's premier institutional brokerage firm. Features:
- ✅ **Modern Animations** — GPU-accelerated Framer Motion
- ✅ **Full Accessibility** — WCAG AA+ compliance
- ✅ **Responsive Design** — Mobile-first Tailwind CSS
- ✅ **Type-Safe** — 100% TypeScript
- ✅ **Rounded Modern UI** — Contemporary button + card styling
- ✅ **Multi-Language** — English & Chinese support
- ✅ **Form UX** — Loading states, validation, error handling

---

## 🏗️ Project Structure (Next.js 16 App Router)

```
src/
├── app/                         # App Router - Pages & Layout
├── components/                  # Reusable Components
├── hooks/                       # Custom React Hooks
├── lib/                         # Utilities & Helpers
├── types/                       # TypeScript Definitions
└── data/                        # Static Data
```

---

## 🚀 Getting Started

```bash
npm install          # Install dependencies
npm run dev         # Start dev server
npm run build       # Build for production
npm start           # Start production server
```
